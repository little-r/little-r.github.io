Tables

Is it true that this is a table?

table(c(1,2,3,4,5,6,7,8))

Yes

because it is a vector of eight values passed as an argument to the function table.

How many columns does t have where t is

table(c(1,1,2,3,3,3))

3

because there are three unique values in the vector passed to table().
